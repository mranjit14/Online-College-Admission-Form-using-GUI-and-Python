# Online-College-Admission-Form-using-GUI-and-Python
This academic group project was developed by second-year Computer Science and Engineering students of St. Thomas' College of Engineering and Technology. The application is built using Python and a GUI framework to streamline the collection and management of newly admitted student information.

The system allows users to:

* Register and store student details for future reference.
* Retrieve and display student records on demand.
* Validate user inputs through field-specific checks, such as 10-digit phone number verification and standard email format validation.
* Maintain data integrity and security through a role-based authentication system with two user types: Office Admin and Student.

The project demonstrates the implementation of GUI-based application development, data validation, user authentication, and student record management using Python.
